ajax(url, method, callback)

Faz um request HTTP de acordo com o método de entrada para a URL. Em seguida, executa o callback de entrada.
Parameters:
Name Type Description
url string A URL onde o método será chamado.
method string O verbo HTTP (POST, GET, etc).
callback string O callback que será chamado após o método.
